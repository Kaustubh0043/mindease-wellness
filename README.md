## 🛠️ Launch Protocol

### 1. Initialize Backend (Spring Boot)
```bash
cd backend
./mvnw spring-boot:run
```

### 2. Initialize Frontend (Vite)
```bash
cd frontend
npm install
npm run dev
```

- **Institutional Key**: `123456`
