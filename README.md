## 🛠️ Launch Instructions

### Prerequisites
- Node.js (v18+)
- Java JDK 17
- Maven

### 1. Initialize Backend
```bash
cd backend
./mvnw spring-boot:run
```

### 2. Initialize Frontend
```bash
cd frontend
npm install
npm run dev
```

### 3. Verification Key
For the demo initialization, use the universal Neural Key: `123456`
